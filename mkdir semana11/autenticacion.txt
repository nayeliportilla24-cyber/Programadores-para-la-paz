El login es el proceso que permite a un usuario ingresar a una aplicación utilizando 
su usuario y contraseña. Esto sirve para comprobar su identidad antes de acceder a la 
plataforma. Algunas rutas deben estar protegidas porque contienen información privada 
o permiten realizar cambios importantes. De esta manera se evita que personas no 
autorizadas entren al sistema. Tampoco se deben mostrar públicamente contraseñas, 
documentos de identidad, teléfonos, direcciones, tokens, claves de API ni otros datos 
sensibles de la comunidad.
