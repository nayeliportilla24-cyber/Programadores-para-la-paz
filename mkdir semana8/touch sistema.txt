El comando uname -a muestra información del sistema operativo, la versión 
del kernel y la arquitectura del equipo.

El comando whoami muestra el nombre del usuario que tiene la sesión activa 
en el sistema.

Conocer esta información es útil al administrar un servidor porque permite 
verificar qué sistema se está utilizando, identificar el usuario con el que 
se trabaja y facilitar tareas de mantenimiento, configuración y seguridad.
