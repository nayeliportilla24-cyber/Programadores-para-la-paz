Backlog priorizado

1. Como ciudadano quiero registrar una incidencia para dejar evidencia del caso.

2. Como coordinador quiero recibir una alerta en Telegram cuando se registre un nuevo reporte.

3. Como miembro del equipo quiero consultar los reportes para hacer seguimiento a cada caso.

4. Como administrador quiero actualizar el estado de los reportes para informar el avance de cada incidencia.

5. Como ciudadano quiero recibir una notificación cuando mi reporte haya sido revisado.
